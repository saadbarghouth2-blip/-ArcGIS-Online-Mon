خطة ArcGIS Online — موقع ثابت (Static Site)
---------------------------
ملفات موجودة في هذا المجلد جاهزة للرفع على أي استضافة ثابتة (ArcGIS Hub, GitHub Pages, Netlify...).

محتويات:
- index.html
- styles.css
- script.js
- ملفات PDF و DOCX المرفوعة (إن وُجدت)
- assets/logo.svg

طريقة الاستخدام:
1. فك الـ ZIP وارفعه على الاستضافة. تأكد أن index.html هو الصفحة الافتراضية.
2. لعرض معاينة PDF داخل الموقع، تأكد أن ملفات الـ PDF موجودة في نفس المجلد.
3. لو عايز إضافات (خرائط مدمجة، صفحات منفصلة لكل أسبوع، أو تحميل تلقائي)، ابعتلي المطلوب وسأعدّل الموقع.
